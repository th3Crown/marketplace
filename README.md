## Setup & Installation

1. **Move Files to HTDOCS**
   - Place all files in `htdocs/marketplace/` (or your desired folder)

2. **Setup Database**
   - Open phpMyAdmin
   - Copy SQL code from `sql/db.txt`
   - Paste and run in the SQL tab
   - Database will be created automatically

3. **Start XAMPP**
   - Start Apache and MySQL

4. **Access the System**
   - Navigate to: `http://localhost/marketplace/`

## How to Use

### For Regular Users
- **Sign Up**: Create new account on login page
- **Dashboard**: View all products
- **Add Listing**: Upload your own products
- **Orders**: View your purchases
- **Notifications**: Get alerts on product approvals

### For Admin
- **Login Credentials**: 
  - Username: `admin`
  - Password: `password`
- **Admin Dashboard**: `http://localhost/marketplace/admin/dashboard.php`
- **Manage Products**: Approve/reject product listings
- **Manage Users**: View and delete users
